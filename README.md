# Hash based Authentication
Hash based authentication system using mongodb and express

This repo will help you to create hash based authentication system. 

# Install dependecies 
After cloning/downloading the repo, at the working directory, issue the following command in the terminal: 

npm install 

# How to run 
In the working directory, issue the following command: 

nodemon server.js 

in the web browser, run the following url: 

localhost:8080/registration.html
